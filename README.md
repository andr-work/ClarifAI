# ClarifAI

Chrome extension that explains unfamiliar words, phrases, and text in simple terms, in English.

## How to use

### Prerequisites

- Google Chrome (Manifest V3 extension support)
- Chrome environment where `LanguageModel` API is available (the extension uses built-in on-device AI APIs)

### Install locally

1. Clone this repository.
2. In Chrome, open `chrome://extensions`.
3. Enable **Developer mode**.
4. Click **Load unpacked**.
5. Select this project folder (`ClarifAI`).

### Use the extension

#### Option 1: Open the dialog box from the extension icon

1. Click the ClarifAI extension icon.
2. The ClarifAI dialog box opens.
3. Enter text and click **Explain**.

#### Option 2: Context menu on selected text

1. Select text on a webpage.
2. Right-click and choose the ClarifAI context menu item.
3. The floating UI appears near the selection and shows the explanation.
